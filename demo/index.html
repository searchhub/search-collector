Search Collector Demo

This example demonstrates the use of the search-collector library.
Checkout the collector-integration.js file for the example implementation based on data-* attributes and the window attached bundle.
For a template using es6 modules and a build tool checkout our blueprint project.

Usage

  1. Search for any search phrase
  2. (Optional) search for a "redirect" to be redirected to a special campaign landing page to test redirect tracking
  3. Click a product
  4. Put it into the basket
  5. Repeat as many times you wish
  6. Proceed to the checkout
  7. Analyze all tracked events
  8. Clear the event data
